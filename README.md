# Kicks - Premium Sneaker Store

**Kicks** is a high-performance, modern e-commerce web application dedicated to sneaker enthusiasts. Built with a focus on premium user experience, high-speed performance, and SEO optimization, Kicks offers a seamless shopping journey from discovery to cart.

## Overview

This project showcases a fully functional storefront integrated with dynamic API data. It features a sophisticated state-managed cart system, a responsive multi-device product gallery, and industry-standard SEO practices.

### Key Features

- **Dynamic Product Routing**: Real-time data fetching using Next.js App Router and Axios.
- **Intelligent Cart System**: Built with **Context API**, handling unique product combinations (ID + Size + Color) without duplication errors.
- **Hybrid Image Gallery**: A custom-engineered gallery that displays a 2x2 grid on desktops and a focus-image-with-thumbnails system on mobile devices for optimal UX.
- **Professional Feedback**: Integrated **Sonner** notifications for non-blocking user feedback on actions like "Add to Cart."
- **SEO & Social Ready**: Optimized with Next.js Metadata API, JSON-LD structured data for Google Rich Snippets, and OpenGraph tags for social sharing.
- **Type-Safe Architecture**: Fully implemented with **TypeScript** to ensure code reliability and easier scaling.

## Tech Stack

- **Framework**: [Next.js 14/15](https://nextjs.org/) (App Router)
- **Language**: [TypeScript](https://www.typescriptlang.org/)
- **State Management**: [React Context API](https://react.dev/learn/passing-data-deeply-with-context)
- **Styling**: [Tailwind CSS](https://tailwindcss.com/)
- **UI Components**: [shadcn/ui](https://ui.shadcn.com/)
- **Data Fetching**: [Axios](https://axios-http.com/)
- **Slider/Carousel**: [Swiper.js](https://swiperjs.com/)
- **Notifications**: [Sonner](https://sonner.stevenly.me/)
- **Icons**: [Lucide-react](https://lucide.dev/)

## Getting Started

To get a local copy up and running, follow these steps:

### 1. Clone the repository

```bash
git clone [https://github.com/mehedi0022/kicks-landing-page.git](https://github.com/mehedi0022/kicks-landing-page.git)
```

### 2. Install dependencies

```bash
npm install
```

### 3. Run the development server

```bash
npm run dev
```

Open http://localhost:3000 with your browser to see the result.
